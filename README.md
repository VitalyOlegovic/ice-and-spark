# ice-and-spark
Examples of Apache Spark usage to work with Ice And Fire data

To run the examples:

    mvn scala:run -DmainClass=it.codegardenroma.Scala01HelloWorld

    mvn scala:run -DmainClass=it.codegardenroma.Scala02ChessPlayers

    mvn scala:run -DmainClass=it.codegardenroma.Spark01ChessPlayers

    mvn scala:run -DmainClass=it.codegardenroma.Spark11Characters
